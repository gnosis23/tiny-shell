tiny-shell
==========
## run by
```bash
make
./tsh
```

## builtin-command
```bash
tsh> quit
tsh> jobs
tsh> bg
tsh> fg
tsh> pwd
tsh> cd <directory>
tsh> environ
```

## other command 
full path should be specified, for example
```bash
tsh> /bin/echo hello world
tsh> /bin/ls -l
tsh> /bin/pwd
```
