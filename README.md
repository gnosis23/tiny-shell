tiny-shell
==========
## run by
```bash
make
./tsh
```

## builtin-command
```bash
tsh> quit
tsh> jobs
tsh> bg
tsh> fg
tsh> pwd
tsh> cd <directory>
tsh> environ
```

## other command 
```bash
tsh> echo hello world
tsh> ls -l
```
